# WC2026 — 2026 世界盃球員排行

2026 世界盃（美加墨主辦，48 隊）的互動式排行網頁，依球員所屬聯賽、俱樂部、身價與「戰鬥力」做視覺化排名。純前端、無後端。

## 開啟方式

直接用瀏覽器雙擊開啟 [`WC2026_Big5_Ranking.html`](WC2026_Big5_Ranking.html) 即可，無需架站或安裝。

## 檔案說明

| 檔案 | 用途 |
|---|---|
| [`WC2026_Big5_Ranking.html`](WC2026_Big5_Ranking.html) | 主頁面，載入下列資料檔後渲染排行 |
| `clubdata.js` | 各隊的俱樂部分布資料 |
| `clubmeta.js` | 俱樂部台灣譯名 + 五大聯賽（英超／西甲／德甲／義甲／法甲）標記 |
| [`WC2026_Big5_Ranking.md`](WC2026_Big5_Ranking.md) | Markdown 版排行表（純文字檢視用） |

> `clubdata.js`、`clubmeta.js` 與 HTML 放在同目錄、由頁面自動載入，搬動時請整包一起搬。
